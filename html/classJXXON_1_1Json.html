Detailed Description

Main object mapper class.

Produced as output of toJson() on serialization. Consumed as constructor parameter for serializable classes on deserilization.

Example:

struct Sample : public JXXON::Serializable
{
Sample()
{
}
Sample(const JXXON::Json &json) : property(json.get<decltype(property)>("property_name"))
{
}
virtual JXXON::Json toJson() const override
{
JXXON::Json json;
json.set("property_name", property);
return json;
}
std::string property;
};
